    "**1. What is the difference between \"Love\" and \"love\" in NLP?**\n",
    "Answer :\n",
    "In NLP, \"Love\" and \"love\" are treated as different words because of uppercase and lowercase letters. This can create confusion for the model and increase the number of unique words. To solve this, we usually convert all text into lowercase. This helps in making the data more consistent and easier to process.\n",
    "\n",
    "**2. What happens if stopwords are not removed?**\n",
    "Answer :\n",
    "If stopwords are not removed, the text will contain many common words like \"is\", \"the\", and \"and\". These words do not add much meaning but increase the size of the data. Because of this, the model may become slower and less efficient. Removing stopwords helps focus only on important words.\n",
    "\n",
    "**3. Provide two real-world scenarios where removing stopwords can be harmful.**\n",
    "Answer :\n",
    "Sometimes removing stopwords can change the meaning of a sentence. For example, in \"I am not happy\", removing \"not\" gives the wrong meaning. Also, in questions like \"What is your name?\", removing \"what\" makes the sentence unclear. So, stopwords should not always be removed blindly.\n",

    "import re\n",
    "\n",
    "def preprocess_text(text):\n",
    "    if not isinstance(text, str) or text.strip() == \"\":\n",
    "        return [], \"\"\n",
    "\n",
    "    # Remove URLs & emails\n",
    "    text = re.sub(r\"http\\S+|www\\S+|https\\S+\", \"\", text)\n",
    "    text = re.sub(r\"\\S+@\\S+\", \"\", text)\n",
    "\n",
    "    # Remove numbers\n",
    "    text = re.sub(r\"\\d+\", \"\", text)\n",
    "\n",
    "    # Convert to lowercase\n",
    "    text = text.lower()\n",
    "\n",
    "    # Handle repeated characters (soooo → soo)\n",
    "    text = re.sub(r'(.)\\1{2,}', r'\\1\\1', text)\n",
    "\n",
    "    # Remove special characters (keep words)\n",
    "    text = re.sub(r\"[^a-z\\s]\", \"\", text)\n",
    "\n",
    "    # Remove extra spaces\n",
    "    text = re.sub(r\"\\s+\", \" \", text).strip()\n",
    "\n",
    "    tokens = text.split()\n",
    "\n",
    "    # Remove short words (<=2) except 'no', 'not'\n",
    "    cleaned_tokens = [\n",
    "        word for word in tokens \n",
    "        if len(word) > 2 or word in [\"no\", \"not\"]\n",
    "    ]\n",
    "\n",
    "    cleaned_sentence = \" \".join(cleaned_tokens)\n",
    "\n",
    "    return cleaned_tokens, cleaned_sentence"

    "def token_analytics(tokens):\n",
    "    total_tokens = len(tokens)\n",
    "    unique_tokens = len(set(tokens))\n",
    "    avg_length = sum(len(word) for word in tokens) / total_tokens if total_tokens > 0 else 0\n",
    "\n",
